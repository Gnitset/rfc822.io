rfc822.io
=========

An ansible setup for The Perfect Mailserver(tm) ;)

Steps to install:
* apt install --no-install-recommends python-pip python-setuptools
* pip install ansible
* ansible-playbook local.yml

* doveadm pw -s SHA512-CRYPT
